WireGuard

WireGuard

1.安装方法

yum update -y 
reboot # unless there were no updates (RTNETLINK answers: Operation not permitted)
sudo curl -Lo /etc/yum.repos.d/wireguard.repo https://copr.fedorainfracloud.org/coprs/jdoss/wireguard/repo/epel-7/jdoss-wireguard-epel-7.repo
sudo yum install epel-release
sudo yum install wireguard-dkms wireguard-tools

2.创建device, 添加ip地址

ip link add dev wg0 type wireguard
ip address add dev wg0 192.168.2.1/24

ip link delte

3.启动服务

wg-quick up wg0
wg-quick up ./wg0.conf
sudo wg show

# enable ipv6
vi /etc/sysctl.conf

net.ipv6.conf.all.disable_ipv6 = 0
net.ipv6.conf.default.disable_ipv6 = 0
net.ipv6.conf.lo.disable_ipv6 = 0

# config kernel parameters at runtime
sysctl -p /etc/sysctl.conf

route del default
sudo route add default gw 35.201.231.236(vps-ip)
sudo route add -host 35.201.231.236 gw 192.168.1.253
