Tor with Meek

Set up tor with meek

dependencies: tor, meek, go

1.Download meek, use proxy in censored regions

git clone https://git.torproject.org/pluggable-transports/meek.git

2.Install go

sudo yum install go

3.Download meek dependency, use proxy in censored regions

git config --global http.proxy socks5h://127.0.0.1:1080
http_proxy=socks5h://127.0.0.1:1080 go get git.torproject.org/pluggable-transports/goptlib.git
git config --global --unset http.proxy

4.Build meek client

cd ~/meek/meek-cilent
go build
sudo cp meek-client /etc/tor/

5.Configure torrc

sudo vim /etc/tor/torrc

minimum torrc configurations:

UseBridges 1
Bridge meek 0.0.2.0:3
ClientTransportPlugin meek exec ./meek-client --url=https://meek.azureedge.net/ --front=ajax.aspnetcdn.com --log meek-client.log

6.Start tor under the dir that has compiled meek-client

tor

一些问题

用systemctl启动的时候需要设置/usr/lib/systemd/system/tor.service的信息否则会报错找不到meek-client的路径。。所以最简单的方法是直接用tor命令启动。
